## When to Create Research

- Exploring a new technology, library, or approach
- Comparing multiple options without a clear winner
- Gathering context before making an architectural decision
- Learning about a domain or API

## Naming Convention

`NN_Topic-Name.md` — Sequential number prefix, kebab-case title.

Examples:
- `01_PDF-Rendering-Options.md`
- `02_Template-Engine-Comparison.md`
- `03_Auto-Fit-Strategies.md`
